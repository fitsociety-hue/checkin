# QR Code Check-in Generator

CSV 형식의 명단을 업로드하여 QR코드를 일괄 생성하고, 문자로 발송하는 웹 애플리케이션입니다.

## 기능
- **CSV 업로드**: 이름, 소속, 전화번호, 차량번호를 포함한 CSV 파일을 업로드하여 데이터를 파싱합니다.
- **QR 생성**: 각 개인별 데이터가 담긴 QR코드를 즉시 생성합니다.
- **문자 발송**: Solapi (CoolSMS) API를 이용하여 생성된 QR코드(MMS)와 안내 문자를 발송합니다.
- **데이터베이스 연동**: 업로드된 데이터를 Google Sheets로 전송하여 저장할 수 있습니다.
- **반응형 디자인**: PC 및 모바일 환경에 최적화된 프리미엄 디자인.

## 배포 방법 (GitHub Pages)

이 프로젝트는 GitHub Pages를 통해 무료로 호스팅할 수 있습니다.

1. 이 프로젝트를 GitHub 저장소에 올립니다 (Push).
2. 터미널에서 다음 명령어를 실행하여 배포합니다:
   ```bash
   npm run deploy
   ```
   (이 명령어가 `build` 폴더를 `gh-pages` 브랜치에 업로드합니다.)
3. GitHub 저장소의 `Settings` > `Pages` 메뉴로 이동합니다.
4. `Build and deployment` 소스가 `gh-pages` 브랜치로 설정되어 있는지 확인합니다.
5. 제공된 URL에 접속하여 확인합니다.

## 로컬 실행 방법

내 컴퓨터에서 실행하려면 다음 명령어를 입력하세요:

```bash
npm run dev
```

## 설정 가이드

### 1. Solapi (문자 발송)
문자 발송 기능을 사용하려면 [Solapi](https://solapi.com/) 계정이 필요합니다.
- **API Key**: 페이지 설정 메뉴에서 입력 (기본값 제공됨)
- **API Secret**: 보안을 위해 페이지 접속 후 설정 메뉴에서 직접 입력해야 합니다.
- **발신번호**: Solapi 대시보드에서 등록된 발신번호만 사용할 수 있습니다.

### 2. Google Sheets (DB 연동)
Google Apps Script를 통해 데이터를 시트에 저장합니다.
- **Web App URL**: 소스 코드(`src/utils/googleSheets.js`)에 사전 설정되어 있습니다.
- **기능**: '작업 공간'의 'DB 저장' 버튼을 클릭하면 현재 리스트가 시트로 전송됩니다.

## CSV 파일 형식
다음과 같은 헤더를 포함해야 합니다:
`이름`, `소속`, `전화번호`, `차량번호`
(웹페이지 내 '템플릿 다운로드' 기능 제공)
